# FarmGeeker 语音交互技术方案

## 产品路线
企业微信语音（1周MVP）→ 小程序（验证后）→ APP（5000+用户）

## 技术架构
用户发语音 → 企业微信(media_id) → 下载amr → ASR转文字 → AI处理 → TTS转语音(可选) → 回复

## ASR选型
| 服务商 | 方言支持 | 价格(元/小时) | 推荐 |
|--------|---------|-------------|------|
| 腾讯云 | 粤语/川话/东北/河南 | 1.8 | ⭐⭐⭐⭐⭐ |
| 讯飞 | 23种方言(最全) | 3.0 | 方言备用 |
| 百度 | 粤语/川话 | 1.5 | 备选 |
| 阿里云 | 粤语/川话 | 2.5 | 备选 |

## 成本：¥12/月/客户
- ASR ¥2.4 + TTS ¥0.62 + AI ¥9

## 交互原则
1. 零打字，全程语音
2. 回复≤3句话，每句≤15字
3. 口语化（"赚了800块"不说"营业额800元"）
4. 主动引导提供选项
5. 听不懂就追问不报错

## 语音日报（每天早上8点，30秒）
问候→昨日营业额→今日待办→库存预警→天气提醒

## 开发排期
- 第1周：企微语音+ASR对接 → MVP
- 第2周：口语化优化+TTS
- 第3周：语音日报+方言测试
- 第4周：小程序(可选)
