# FarmGeeker 成本核算与定价调整分析

## 核心发现

1. **原定价（199/399/699）全线亏损** — 人力成本是大头，占47%-97%
2. **AI 模型成本差异巨大** — Claude Sonnet ¥291/月 vs 通义千问 ¥2.4/月（122倍）
3. **主动智能成本可控** — 仅增加 8-16% 的 AI 成本
4. **必须涨价 + 分层模型策略**

## 关键策略

1. **分层模型** — 基础/标准用通义千问（¥2/月），高级用 Claude Sonnet（¥291/月）
2. **开通费** — 覆盖数据接入一次性成本
3. **超量收费** — 主动分析次数限制，超量付费增收 15-20%
4. **老客户过渡** — 3个月过渡期，新客户新价格
5. **规模效应** — 人力成本是大头，200客户是盈利拐点
